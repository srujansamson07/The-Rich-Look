# TheRich Look E-Commerce Demo

Static luxury menswear storefront for TheRich Look.

## Edit Products

Update product names, prices, descriptions, categories, sizes, colors, and image paths in:

`products/products.js`

Product images live in:

`images/products/`

## Pages

- `index.html` - home page
- `shop.html` - searchable/filterable shop
- `product.html?id=product-id` - product detail page
- `cart.html` - cart and totals
- `checkout.html` - customer details and email order flow
- `about.html` - brand story
- `contact.html` - contact form and map

## Notes

This is a front-end demo. Checkout opens the visitor's email app with the order details filled in. A real production checkout can later be connected to Shopify, WooCommerce, Razorpay, Stripe, or a custom backend.
